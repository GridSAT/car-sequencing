car-sequencing
==============

CNF generator for car-sequencing problems from http://www.cs.st-andrews.ac.uk/~ianm/CSPLib/prob/prob001/

Version info:

```bash
./gen_sat -help
```
Help:

```bash
./gen_sat -help
```

To generate models, e.g. model E3 with symmetry breaking:

```./gen_sat -file hard/p00.txt -e3 -sym
